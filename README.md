## ⚡ Bawaal Features
* **Genomic Cinematic DNA Mapping:** An avant-garde analytical engine that parses user watch patterns, genre frequencies, and interaction metrics to generate a localized cryptographic "Cinematic DNA" signature for absolute personalization profiles.
* **Real-Time Dynamic Battles:** A competitive social layer allowing registered users to lock vectors and engage in real-time cinematic trivia, watchlist comparisons, and engagement battles to determine the ultimate cinema connoisseur leaderboard standing.
* **Dynamic TMDB Sync:** Real-time global movies fetching, advanced search indices, metadata loading, and crisp asset streaming.
* **Executive Session Security:** State-of-the-art secure session mapping with strict `Lax` SameSite cookie boundaries.
* **Dual-State Storage:** Transparent syncing between anonymous visitor state tracking and registered user accounts.
* **Advanced Analytics Pipeline:** Persistent logging for search histories and watch status classifications (`watchlist`, `watching`, `watched`, `favorites`).
* **Responsive Premium UI:** Fully dynamic frontend optimized for high visual fidelity and ultra-smooth navigation transitions.

## 🛠️ High-Performance Architecture Stack
* **Genomic Cinematic DNA Mapping (DNA Analysis):** An avant-garde analytical engine that parses user watch patterns, genre preferences, and interaction metrics to generate a localized cryptographic "Cinematic DNA" signature for absolute personalization profiles.
* **Real-Time Interactive Watch Battles (Dynamic Battles):** A competitive social layer allowing registered users to lock vectors and engage in real-time cinematic trivia, watchlist comparisons, and engagement battles to determine the ultimate cinema connoisseur leaderboard standing.
* **Frontend:** Semantic HTML5, Advanced Layout CSS3, Native Vanilla JavaScript (ES6+)
* **Backend Core:** PHP 8.2 (Secure Routing, Modular Architecture APIs Pipeline)
* **Database Cloud Layer:** MySQL 8.0 / InnoDB Relational Integrity Engine
* **External Integrations:** The Movie Database (TMDB) RESTful Framework API
